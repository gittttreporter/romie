Alpha Testing - Bug Report Instructions

When you find an issue, please help me debug by sharing logs:

Windows

1. Press Win + R
2. Type: %APPDATA%\romietron\logs
3. Press Enter
4. Attach the latest .log file to your bug report

Mac

1. Press Cmd + Space
2. Type: ~/Library/Application Support/romietron/logs
3. Press Enter
4. Attach the latest .log file to your bug report

What to Include

- Steps to reproduce the issue
- What you expected vs what happened
- The log file from above
- Screenshots if helpful

That's it! The logs will help me track down exactly what went wrong.
