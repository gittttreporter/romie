ROMie Alpha - Installation & Bug Reporting

INSTALLATION WARNINGS

Since I'm an unverified publisher, you'll see security warnings:

Windows:
- "Windows protected your PC" dialog will appear
- Click "More info" then "Run anyway"
- This is normal for unsigned software

Mac:
- "ROMie can't be opened because it's from an unidentified developer"
- Right-click the app and select "Open"
- Click "Open" in the confirmation dialog
- Or go to System Preferences > Security & Privacy and click "Open Anyway"

BUG REPORT INSTRUCTIONS

When you find an issue, please help me debug by sharing logs:

Windows

1. Press Win + R
2. Type: %APPDATA%\ROMie\logs
3. Press Enter
4. Attach the latest .log file to your bug report

Mac

1. Press Cmd + Space
2. Type: ~/Library/Application Support/ROMie/logs
3. Press Enter
4. Attach the latest .log file to your bug report

What to Include

- Steps to reproduce the issue
- What you expected vs what happened
- The log file from above
- Screenshots if helpful

That's it! The logs will help me track down exactly what went wrong.
